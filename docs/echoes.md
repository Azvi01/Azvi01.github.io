# Ecos (Echoes)

Los **Echoes** son criaturas cuya energía puedes absorber para obtener habilidades.

## Tipos de ecos
- Curación
- Daño elemental
- Buffs
- Movilidad
- Transformaciones

## Rarezas
- 1★ → Ecos básicos  
- 4★ → Ecos avanzados  
- 5★ → Ecos élite  

Cada eco viene con:
- Artefacto
- Habilidad activa única
- Bonus de stats
