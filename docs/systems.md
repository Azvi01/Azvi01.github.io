# Sistemas Principales

## 1. Movimiento y Exploración
- Doble salto  
- Carreras verticales  
- Grapple (enganche rápido)  
- Deslizamiento dinámico  
- Escalada libre  

## 2. Combate
- Ataques básicos y cargados  
- Parry perfecto  
- Esquiva con tiempo bala  
- Habilidades de resonancia  
- Ultimates de cada personaje  

## 3. Echoes
Un sistema que permite absorber criaturas para obtener:
- Pasivas
- Estadísticas
- Habilidades activas especiales

## 4. Progresión
- Subida de nivel
- Ascensiones
- Refinar armas
- Eco-builds
